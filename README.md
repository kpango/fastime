Blazing Fast Time Library for golang with zero allocs

## Requirement
Go 1.11

## Installation
```shell
go get github.com/kpango/fastime
```

## Example
```go
    now := fastime.Now()
    defer fastime.Stop()

    // Create Instance
    ft := fastime.New()
    defer ft.Stop()
    ft.Now()
```
